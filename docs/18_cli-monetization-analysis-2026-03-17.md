- $5/月: 手数料 $0.45、実質収益 $4.55（手数料率 9%）

## 業界の動向

### Tailwind Labs の崩壊（2025-2026）
- AI コーディングアシスタントの台頭でドキュメントトラフィック 40% 減少
- 売上 80% 下落、エンジニア 75% レイオフ
- テンプレート/ドキュメント販売型ビジネスは AI 時代に脆弱

### CLI ツールの課金パターン

課金に成功した CLI ツール（ngrok, Railway, Warp）は全て「CLI はインターフェース、価値はサーバーサイド」。
純粋なローカル CLI ツールの課金成功例:
- ESLint: スポンサーシップのみ（課金モデルなし）
- Prettier: スポンサーシップのみ
- Biome: VC 資金で運営 → 破綻
- Fig: 買収（単独での課金は未達成）

### Caleb Porzio（Livewire 作者）の Sponsorware モデル
- GitHub Sponsors で累計 $1M+ 達成
- 535 人が $14/月以上を支払い
- 戦略: スポンサー限定で新機能を先行公開 → 閾値到達後に OSS 化
- 実装コストほぼゼロ、CLI 認証不要

## 推奨アクション（優先順位順）

### Option A: Sponsorware（Caleb Porzio 型）— 推奨
- 新機能をスポンサー限定で先行公開
- 期待収益: $1-3K/月
- リスク: 低
- 実装コスト: ほぼゼロ
- 前提: まず GitHub Stars 500+ が必要

### Option B: プラットフォーム化（Vercel/Netlify 型）
- CLI は 100% 無料（認証不要、制限なし）
- Web ダッシュボードを有料化: チーム共有、コンテキスト履歴、分析
- $15-20/席/月（チーム向け）
- 期待収益: $5-20K/月
- リスク: 高（開発 6-12 ヶ月）

### Option C: 完全 OSS + スポンサーシップ
- GitHub Sponsors / Open Collective
- 期待収益: $0.5-5K/月（10K+ stars 到達時）
- リスク: 低

### Option D: ポートフォリオ/コンサル活用
- Orbit を実績として「AI 開発ワークフローコンサルティング」
- $200-400/時間、$5-15K/案件
- 期待収益: $50-150K/年（2-3 クライアント）

### Option E: 別のプロダクトに注力
- AI コンテキスト生成市場は AI アシスタント自体の進化で消える可能性
- Orbit に費やした技術力を別の問題に向ける
